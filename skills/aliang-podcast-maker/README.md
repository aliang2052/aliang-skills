# aliang-podcast-maker · 多人对话播客制作

## 干什么

把**多角色对话脚本**做成一条完整的语音播客：

1. 分析对话，识别角色，为每个角色分配合适音色
2. 逐句生成语音（每句一个文件）
3. 用 `ffmpeg` 按对话顺序拼接成完整音频
4. 可选加片头/片尾音效

适合访谈、对谈、剧本朗读等多人场景。

## 如何安装

需先安装百炼 CLI（见[仓库根 README](../../README.md#-前置依赖安装百炼-cli)）和 `ffmpeg`（拼接音频用），然后：

```bash
cp -R aliang-podcast-maker ~/.claude/skills/
```

## 如何使用

把对话脚本丢给你的 AI 助手，例如：

```
把下面这段对话做成播客（阿琳活泼、老陈沉稳）：
阿琳：AI 这么猛，到底还需要什么样的人才？
老陈：我觉得是 T 字型的人……
```

AI 会先给音色方案让你确认，再逐句配音、拼接成完整 mp3。

> 用系统音色（如 `longanhuan` 欢脱女、`longtian_v3` 磁性男），**无需克隆**，开箱即用。
> 注意：百炼 TTS 输出虽是 `.mp3` 后缀但实为 WAV，拼接时会自动转码为真 MP3。
