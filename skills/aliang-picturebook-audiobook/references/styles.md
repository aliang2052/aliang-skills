# 画风预设与智能推荐

`build_audiobook.py --style <名称>` 直接传下面的预设名即可（脚本会展开成完整提示词后缀）；传任意其他文字则当作自定义画风。

## 预设一览

| 预设名 | 风格 | 最适合的题材 |
|---|---|---|
| `水彩绘本` | 柔和水彩、温暖治愈（通用默认） | 日常、情感、动物、亲子 |
| `国潮插画` | 鲜艳撞色、描金线条、祥云纹样 | 中国神话/英雄（哪吒、悟空、年兽） |
| `国风水墨` | 水墨工笔、淡彩留白、古典雅致 | 古诗词、历史、传统寓言、二十四节气 |
| `剪纸民俗` | 红金剪纸、年画喜庆、对称纹样 | 春节、民俗、十二生肖、节日 |
| `皮克斯3D` | 3D卡通、立体圆润、电影质感 | 现代冒险、科幻、机器人、太空 |
| `蜡笔手绘` | 蜡笔稚拙、童真涂鸦 | 低龄(3-5岁)、情绪启蒙、幼儿园日常 |
| `扁平矢量` | 几何扁平、明快撞色 | 科普、认知、职业体验、习惯养成 |

## 智能推荐规则（根据故事内容选默认）

按题材匹配，推荐一个默认值，同时把全部选项给用户挑：

- 中国神话 / 古风英雄（哪吒、悟空、神仙妖怪）→ **国潮插画**（动感鲜艳，最对小朋友胃口）
- 古诗词 / 历史 / 传统寓言 → **国风水墨**
- 春节 / 节日 / 生肖 / 民俗 → **剪纸民俗**
- 科幻 / 太空 / 机器人 / 现代冒险 → **皮克斯3D**
- 科普 / 认知 / 习惯养成 → **扁平矢量**
- 低龄情绪启蒙（3-5岁）→ **蜡笔手绘**
- 其它 / 日常 / 动物 / 拿不准 → **水彩绘本**（通用兜底）

用 AskUserQuestion 把推荐项放第一个并标「(推荐)」，让用户确认或改选。
