## 通用爽点节奏原则

无论哪个题材，都遵守：

1. **黄金 3 秒**：第一镜就抛冲突/悬念/反差，不铺垫。短剧没有耐心成本。
2. **钩子前置**：每 15-20 秒一个小反转或情绪爆点，结尾留扣。
3. **情绪过山车**：压抑 → 蓄力 → 爆发，落差越大越爽。
4. **爽点要"被看见"**：打脸/逆袭必须有"众人见证"的公开场合，不能私下完成。
5. **每次随机抽取**：从对应题材爽点库随机选 8-12 个组合，避免套路雷同（可用 `scripts/pick_beats.py` 抽签）。

## 各题材爽点库与结构

### 草根逆袭

**爽点库（随机抽 8-12）**：低谷开局、被轻视、天赋觉醒、贵人相助、破釜沉舟、技能碾压、身份反转、绝地反击、一战成名、众人见证、打脸羞辱、知遇之恩、回馈社会、坚持不懈、逆风翻盘、扮猪吃虎、实力打脸、草根登顶。

**逆袭路径（随机选 1-2，拒绝"编程创业上市"）**：技艺类（厨师/调酒/修复师/手艺人）、艺术类（画家/音乐/舞者）、体育类（拳击/电竞）、传统类（非遗/老字号）、新兴类（直播/无人机/AI 训练师）、服务类（收纳/婚礼策划/宠物训练）、特殊类（侦探/鉴宝/谈判）。

**结构**：低谷被轻视 → 偶然展露/觉醒 → 拜师或自我打磨 → 遭遇打压挫折 → 善举埋伏笔 → 贵人/机遇 → 关键擂台 → 公开碾压打脸 → 圆满+回馈。

### 复仇打脸

**爽点库**：含冤受辱、隐忍蛰伏、收集证据、扮猪吃虎、身份隐藏、实力反转、步步紧逼、当众揭穿、对手溃败、墙倒众人推、真相大白、正义伸张、化敌为友、自我救赎、强者归来。

**结构**：开局被陷害/羞辱 → 跌入谷底立誓 → 暗中积蓄力量（隐藏身份） → 重回原局众人不识 → 逐个击破对手 → 高潮当众亮明身份/证据 → 反派连锁崩塌 → 受害者平反 → 主角放下或升华。

**注意**：复仇要"正义"，不血腥、不私刑，用法律/规则/实力让反派自食其果。

### 都市甜宠 / 霸总言情

**爽点库**：命运相遇、误会反差、强势守护、吃醋占有、双向奔赴、反差萌、高调宠溺、危难相救、身份悬殊、众人羡慕、破除阻力、当众官宣、家人认可、破镜重圆、跨越鸿沟。

**结构**：意外相遇（冲突或误会）→ 强制同框/契约关系 → 互动升温（2-3 次帮助/守护）→ 误会或外部阻力 → 一方主动打破隔阂 → 高光告白/守护 → 排除万难 → 当众官宣圆满。

**红线**：感情要"因了解而爱"，必须有相遇契机 + 2-3 次互动铺垫，禁止"看到成功就爱上"。

### 悬疑反转

**爽点库**：开场悬念、信息差、误导铺垫、抽丝剥茧、关键反转、身份反转、真凶意外、伏笔回收、细思极恐、绝境破局、智斗博弈、真相大白、多重反转、闭环收尾。

**结构**：抛出反常事件/谜题 → 主角介入调查 → 第一层表象答案 → 推翻、出现新线索 → 中段大反转（怀疑对象错） → 信息差揭开 → 终极反转 → 伏笔全收、闭环。

**注意**：反转必须前文埋有伏笔，可回看验证；不能为反转而反转。

### 家庭伦理

**爽点库**：误解积怨、隐忍付出、真相浮现、亲情羁绊、舍己为家、知错和解、守护承诺、代际理解、雪中送炭、苦尽甘来、团圆相聚、宽容大度、感恩回报。

**结构**：家庭矛盾爆发 → 一方默默承受 → 误会加深至临界 → 关键事件揭开隐情 → 互相理解 → 共渡难关 → 和解团圆 → 情感升华。

### 古装重生 / 穿越

**爽点库**：重生归来、先知优势、改写命运、宿敌再遇、步步为营、扮猪吃虎、打脸前世仇人、身份反转、宫斗/商战智胜、护住所爱、因果轮回、强势翻盘、众人惊叹。

**结构**：含恨而终/穿越开局 → 重生回关键节点 → 利用先知布局 → 提前破解前世劫难 → 与宿敌交锋 → 关键反杀打脸 → 守住所爱之人 → 改写结局。

### 战神赘婿 / 兵王（男频）

**爽点库**：隐藏身份、被轻视、绝境亮剑、实力碾压、一招制敌、身份揭晓、众人震惊、强者云集相迎、打脸豪门、守护家人、扮猪吃虎、王者归来、一言九鼎。

**结构**：低位身份被欺辱（赘婿/落魄）→ 隐忍 → 危机触发出手 → 小试身手露锋芒 → 各方势力登场试探 → 真实身份层层揭开 → 高潮当众碾压 → 打脸所有看轻者 → 守护与回归。

**注意**：燃但不暴力血腥，冲突用「实力威慑/规则碾压」替代打斗细节。

### 搞笑沙雕

**爽点库**：反差人设、误会连环、神转折、自作自受、装到翻车、歪打正着、群体捧哏、打破第四面墙、反差萌、意外救场、笑中带泪、无厘头逆袭。

**结构**：荒诞设定开场 → 主角离谱操作 → 误会滚雪球 → 接连翻车 → 阴差阳错立功 → 真相爆笑揭开 → 圆满收场（可带一点暖）。

## 反套路红线

- ❌ 「自学编程 → 创业 → 上市」
- ❌ 「救老人 → 老人是富豪 → 给投资」
- ❌ 「一夜暴富 / 突然天降资源」
- ✅ 逆袭路径新颖、成功多元（不止财富）
- ✅ 转机来自前文铺垫的善举/努力，有因果

## 内容健康红线

| 类别 | 禁止 | 替代 |
|------|------|------|
| 血腥暴力 | 流血、厮杀、殴打细节 | 用冲突、对峙、威慑 |
| 身体伤害 | 溃烂、伤疤、残疾 | 用辛苦、劳损、磨砺 |
| 负面极端 | 绝望、想死、崩溃 | 用迷茫、低谷、挣扎 |
| 违规导向 | 私刑、犯罪美化 | 用法律/规则伸张正义 |
